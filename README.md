# ML Fundamentals

Hands-on machine learning practice spanning classical ML, neural networks, NLP, and reinforcement learning. Built while learning core concepts and applying them to real datasets and Kaggle competitions.

## Contents

| Notebook | Topic | Stack |
|---|---|---|
| `Copy_of_seaborn.ipynb` | Data visualization | seaborn, matplotlib |
| `ClassicalML.ipynb` | Linear regression, logistic regression, SVM | scikit-learn |
| `KNN_and_Decision_Trees.ipynb` | kNN and decision tree classification (wine dataset) | scikit-learn, GridSearchCV |
| `Neural_Nets_Implementation.ipynb` | Neural network for binary classification (breast cancer dataset) | PyTorch |
| `embedding_models.ipynb` | Text embeddings | Hugging Face transformers, PyTorch |
| `GloVe_and_BERT_Clustering_and_Sentiment_Analysis.ipynb` | Text clustering and sentiment analysis | GloVe, BERT, KMeans |
| `ML_Based_Chatbot.ipynb` | Intent-based chatbot | TF-IDF, logistic regression |
| `Reinforcement_learning.ipynb` | RL agent training | OpenAI Gym |
| `Using_Neural_Nets_on_Kaggle_Competition.ipynb` | CNN for Kaggle competition (+ `train.csv`, `test.csv`) | PyTorch |
| `Kaggle_Competition_Submission.ipynb` | Spaceship Titanic — survival classification | scikit-learn, BayesSearchCV |

## Topics Covered

- **Classical ML**: linear/logistic regression, SVM, kNN, decision trees, random forests
- **Model tuning**: GridSearchCV, BayesSearchCV
- **Deep learning**: feedforward neural nets, CNNs (PyTorch)
- **NLP**: TF-IDF, word embeddings (GloVe), transformer embeddings (BERT, Hugging Face), sentiment analysis, intent classification
- **Reinforcement learning**: agent training in OpenAI Gym environments
- **Data visualization**: seaborn, matplotlib

## Notes

Notebooks were originally developed in Google Colab; outputs are preserved where relevant (plots, training logs, sample runs).
